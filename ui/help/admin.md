# Admin Portal Instructions

This page is for organizers and data managers. Volunteers do not typically see it.

## Main Action Buttons
- **Open Contact Review Dashboard**: Review voter contacts and pulse opt-ins with keyboard-friendly navigation.
- **Manage Icebreakers & Scripts**: Update volunteer scripts and campaign touchpoints.
- **Manage Volunteers**: Maintain volunteer records, access emails, and active status.
- **Field Support Desk**: Monitor active field sessions, location freshness, nearby voters, and live tasks.

## Legacy Contact Controls
- **Refresh**: Reloads legacy contacts with your current filter and page.
- **Filter**: Switch between all, unreviewed-only, and reviewed-only records.
- **Prev / Next**: Paginates through legacy contacts.
- **View**: Opens full record JSON/details.
- **Edit**: Opens modal editor for contact fields.
- **Delete**: Permanently deletes the selected contact record.
- **Save Changes**: Commits edits immediately.
- **Cancel**: Closes edit modal without saving.

## Field Support Desk Buttons
- **Refresh**: Reload active sessions and tasks.
- **Load voters on this street**: Pull nearby voters based on field volunteer street hint/GPS.
- **Add call-ahead / Add next stop**: Create support tasks for the active field volunteer.
- **Confirm / Done / Cancel**: Update task status.
- **Call ahead / Next stop** (nearby voter rows): Create prefilled tasks from voter records.

## Access & Safety
- All actions require Cloudflare Access plus the `ADMIN_EMAILS` allow-list.
- Double-check before editing voter data; changes are live.
- Use the help toggle here for tips specific to each admin subpage.
