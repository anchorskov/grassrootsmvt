# Volunteer Intake

Use this page to add a new volunteer or request updates. Everything goes to a **staging table** first; an admin reviews before changes take effect.

- Add new volunteer: Leave “Update” off and submit the new info.
- Check my info: Click **Check my info** to load your own record. Edit and submit to request updates to your entry only.
- Update existing volunteer (admins only): Choose “Update” and pick a volunteer from the dropdown to prefill their info.
- Mark as active: Turn on for active/available volunteers; off for inactive/paused.
- State/City: Helps with coordination; state defaults to WY.
- What happens next? Admins approve/reject staging records, then apply changes to the main volunteers table.

Tips
- Email should match the volunteer’s Cloudflare Access email.
- Phones are normalized to E.164 (+1…).
- Keep notes brief and actionable (role, availability, corrections).
