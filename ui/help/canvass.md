# Canvass Page Instructions

Use this page for door-to-door canvassing, nearby voter lookups, and real-time support coordination.

## Quick Start

1. **Set targeting filters** from the Hub before opening this page.
2. **Optional but recommended: open GPS Coordination first** if you have a support contact.
3. **Enter Street, then House #** using autocomplete.
4. **Set range/limit** and press **Find nearby**.
5. **Log outcomes** on each voter card as you work down the street.

## Button Help

- **Enable GPS**: Expands the GPS Coordination panel.
- **Start GPS Session**: Opens the field-session page in a new tab so you can share location and receive tasks.
- **Copy**: Copies the support desk URL to your clipboard.
- **Find nearby**: Loads nearby voters using your selected address and filters.
- **📋 Contact**: Opens the full contact workflow for a complete interaction record.
- **🚪 Not Home**: Quick-logs that no one answered.
- **📝 Note**: Opens the update form to capture notes and follow-up details.
- **Call**: Opens the phone workflow for that voter when a number is available.

## GPS Coordination Flow

1. Expand **GPS Coordination**.
2. Press **Start GPS Session**.
3. In the new tab, share your location.
4. Send your support contact the desk URL using **Copy**.
5. Support can assign call-ahead and next-stop tasks while you canvass.

## Canvass Logging Tips

- Enter **Street first**, then **House #**.
- Keep **range** small for dense blocks to reduce noise.
- Use **Contact** for full records and consent updates.
- Use **Not Home** for fast pass-through logging.
- Use **Note** when details matter but full contact flow is not needed.

## Pulse Consent Script

"With your permission we can send brief campaign updates by text or email. Message/data rates may apply and you can reply STOP any time. Is that okay?"

If they agree, record updated phone/email and opt-in in the contact flow.
